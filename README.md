## pickem
Football Pickem App

## Description
This is a NFL weeekly pickem website. I created this website for my co workers who wanted an easy way to have a weekly pickem' games. On this site you will enter your name and e-mail and below that will be a list of NFL matchups and each game will have a radio button to select which team you would like to choose to win for the week.
